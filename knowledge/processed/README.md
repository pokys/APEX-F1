# Processed Signals Specification

This directory contains structured soft-data signals consumed by:
- `pipeline/build_features.py`
- `pipeline/update_ratings.py`

These files are human-curated outputs from article analysis.

## File Naming

Recommended filename pattern:
- `signals_YYYY-MM-DD.json`

The validator accepts any `*.json`, but warns on non-standard names.

## Accepted Top-Level JSON Shapes

Per file, one of:
1. `[{...}, {...}]`
2. `{"signals": [{...}, {...}]}`
3. `{...}` (single signal object)

## Required Fields Per Signal

Required by validator:
- `source_name` (non-empty string)
- `article_hash` (non-empty string)
- `source_confidence` (number in `[0, 1]`)
- At least one of:
  - `team` (non-empty string), or
  - `driver` / `driver_name` (non-empty string)

Conditional:
- If `upgrade_detected == true`:
  - `upgrade_magnitude` must be one of: `minor`, `medium`, `major`

## Optional Fields (Validated Ranges)

- `upgrade_component` (non-empty string if present)
- `reliability_concern` in `[0, 1]`
- `driver_confidence_change` in `[-1, 1]`
- `extraction_version` (non-empty string if present)

## Optional Advanced Fields (Used by Ratings, Not Strictly Required)

These are consumed by rating updates when provided:
- `wet_performance_index` in `[0, 1]` (team-level helper)
- `safety_car_reaction` in `[0, 1]` (team strategy helper)
- `new_component_penalty` in `[0, 1]` (reliability penalty helper)

## Article Hash

`article_hash` should match the collector convention:
- `sha256(normalized_title + normalized_url)`
- normalization = trim + collapse whitespace.

Practical recommendation:
- Keep the hash generated by your extraction workflow/tool.
- If unavailable, use a stable deterministic hash and keep it unchanged.

## Example (Engine Upgrade + Reliability Concern)

```json
{
  "signals": [
    {
      "team": "Ferrari",
      "upgrade_detected": true,
      "upgrade_component": "power unit",
      "upgrade_magnitude": "medium",
      "reliability_concern": 0.22,
      "source_confidence": 0.87,
      "source_name": "autosport",
      "article_hash": "a5e30f6f8a2d2b0c9a9f4b5d9d42c7f0f3754e7e80d8f3a28c2d66df76c1a901",
      "extraction_version": "v1"
    },
    {
      "driver": "LEC",
      "driver_confidence_change": 0.15,
      "source_confidence": 0.82,
      "source_name": "the-race",
      "article_hash": "29f7a4a4e7d6cb7a95e9cf0e598af8a7367bbf6e8de2e7f538c8e278d6824f52",
      "extraction_version": "v1"
    }
  ]
}
```

## Validation Command

```bash
python pipeline/validate_signals.py --signals-dir knowledge/processed --log-level INFO
```

Allow empty directory:

```bash
python pipeline/validate_signals.py --signals-dir knowledge/processed --allow-empty --log-level INFO
```
